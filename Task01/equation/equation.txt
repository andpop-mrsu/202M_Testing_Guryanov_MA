# Тестовые кейсы для функции solve_quadratic
1. a = 1, b = -3, c = 2 → Ожидаемый результат: [1.0, 2.0] (два действительных корня)
2. a = 1, b = 2, c = 1 → Ожидаемый результат: [-1.0] (один действительный корень)
3. a = 1, b = 0, c = -4 → Ожидаемый результат: [-2.0, 2.0] (два действительных корня)
4. a = 1, b = 0, c = 4 → Ожидаемый результат: [] (нет действительных корней)
5. a = 0, b = 2, c = 3 → Ожидаемый результат: Ошибка (не является квадратным уравнением)
6. a = 0, b = 0, c = 0 → Ожидаемый результат: Ошибка (не является квадратным уравнением)
